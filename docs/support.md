# Awake Support

Awake is a macOS menu bar utility that keeps your Mac awake while local AI coding
agents and long-running tasks are active.

## Help

- Open Awake from the menu bar.
- Choose a timed session for manual use.
- Enable supported agent watching to keep the Mac awake while local agent
  activity is detected.
- Use the More window for launch-at-login and app settings.

The Mac App Store version uses public macOS power assertions and does not include
closed-lid support directly. Closed-lid support is available only through a
separate direct-download helper:

https://github.com/adiab98/Awake/blob/main/docs/closed-lid.md

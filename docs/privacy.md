# Awake Privacy Policy

Awake does not collect, sell, share, or upload personal data.

Awake runs locally on your Mac. It watches supported local AI coding tool session
folders and process activity so it can keep macOS awake while work is still in
progress. Awake only uses local file activity timestamps and local process
signals for this purpose.

Awake does not transmit transcript contents, file names, prompts, source code, or
usage analytics to the developer or to third parties.

Awake stores preferences locally in macOS user defaults for the app bundle ID
`com.diabdiab.awake`.

Contact: open an issue at https://github.com/adiab98/Awake/issues
